## ✏ 프로젝트 소개
기존에 나와있는 Weather API 를 사용해서 만든 날씨앱 입니다.  
처음으로 api를 사용해서 만든 프로젝트입니다.  
**영어**로 도시 이름을 입력하면 해당 도시의 날씨가 나타납니다.  
jQuery에서 ajax를 사용하였고, 현재온도, 최고온도, 최저온도, 체감온도 네가지 정보가 나옵니다.

## 🖱 사용 언어
HTML, CSS, jQuery(ajax)
